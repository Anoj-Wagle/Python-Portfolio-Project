# Python Portfolio project
This project aims to provide insights about different python project using pandas, matplot and many more python codes.

# A. Data Job Overview
This project provides an insights into the top country, skills, title of the job, and full time contract realted to data jobs.

# Dataset Overview
The dataset is extracted from **[Hugging Face](https://huggingface.co/datasets/lukebarousse/data_jobs)**
It consists:
* Job Title
* Job Type
* Job Location
* Job Skills
* Job Posted
* Salary Year Average
* Salary Hour Average

# Data Cleaning
- Convert data from train to pandas dataframe.
- Remove Duplicates
- Remove nan values with median values
- Remove unwanted columns
